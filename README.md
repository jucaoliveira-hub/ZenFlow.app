# ZenFlow.app
A minimalist productivity Web App (Kanban + Pomodoro) designed to reduce cognitive load and help ADHD minds focus. Built with Vanilla JS.


# 🌊 ZenFlow - Foco e Clareza

> **"Respire. Uma coisa de cada vez."**

O **ZenFlow** é uma aplicação web minimalista focada na organização de tarefas e produtividade, desenhada especificamente para mentes agitadas e pessoas com TDAH. O projeto nasceu da necessidade de reduzir a sobrecarga cognitiva através de um layout limpo e regras de limitação de trabalho simultâneo.

---

## 🎯 O Problema vs. A Solução

Aplicativos de "To-Do" tradicionais costumam gerar ansiedade ao exibir listas infinitas de tarefas. O ZenFlow combate isso unindo a metodologia Kanban com regras estritas de saúde mental:

- **Limite de Carga Cognitiva:** O sistema emite um alerta e impede que o usuário coloque mais de 2 tarefas na coluna "Em Foco" simultaneamente.
- **Gestão de Energia, não apenas de Tempo:** As tarefas recebem tags de energia (Baixa 🧘, Média ⚡, Alta 🔥), permitindo que o usuário escolha o que fazer com base na sua disposição atual.
- **Pomodoro Nativo:** Um cronômetro integrado de 25 minutos com alerta sonoro garante que o usuário mantenha o ciclo de trabalho e descanso (Focus/Break).

## 🚀 Funcionalidades Principais

* **Quadro Kanban Inteligente:** Colunas dinâmicas (A Fazer, Em Foco, Feito) com contadores em tempo real.
* **Filtros Rápidos:** Separação imediata de tarefas entre "Trabalho" (Azul) e "Pessoal" (Verde).
* **Quotes Motivacionais:** Frases dinâmicas geradas a cada acesso para ancorar o estado de presença.
* **Smart Input:** Adição rápida de tarefas com descrição, data e nível de energia diretamente no painel superior.

## 💻 Tech Stack

O projeto foi construído focado em leveza, performance e acessibilidade, utilizando tecnologias puras sem dependência de frameworks externos:

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/javascript-%23F7DF1E.svg?style=for-the-badge&logo=javascript&logoColor=black)

- **Vanilla JS:** Motor de lógica para manipulação do DOM e do Cronômetro.
- **LocalStorage API:** Persistência de dados diretamente no navegador do usuário, garantindo privacidade total e carregamento instantâneo.
- **CSS Grid & Flexbox:** Arquitetura de layout responsiva e fluida.

## 🔧 Como Utilizar (Localmente)

1. Clone este repositório: `git clone https://github.com/jucaoliveira-hub/zenflow.git`
2. Abra a pasta do projeto e clique duas vezes no arquivo `index.html`.
3. Adicione sua primeira tarefa e inicie o cronômetro para focar!
